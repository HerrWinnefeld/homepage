Was ist Flexbox?

Flexbox ist eine Layout-Technik in CSS, mit der du einfach flexible Layouts erstellen kannst. Mit Flexbox kannst du Elemente auf deiner Seite ausrichten und verteilen, ohne komplizierte Positionierungstechniken zu verwenden.

Flexbox-Beispiel

Hier ist ein einfaches Beispiel, wie Flexbox funktioniert. Drei Boxen werden in einer Zeile gleichmäßig verteilt.

HTML-Code:


<div class="flex-container">
    <div class="box">Box 1</div>
    <div class="box">Box 2</div>
    <div class="box">Box 3</div>
</div>

CSS-Code:


.flex-container {
    display: flex;
    justify-content: space-around; /* Verteilung der Boxen */
    background-color: lightgray;
    padding: 20px;
}

.box {
    background-color: coral;
    padding: 10px;
    width: 100px;
    text-align: center;
}

Wichtige Flexbox-Eigenschaften

  • display: flex; – Aktiviert Flexbox für ein Element.
  • justify-content – Bestimmt die horizontale Ausrichtung der Elemente (z.B. space-around, center, space-between).
  • align-items – Bestimmt die vertikale Ausrichtung der Elemente.
  • flex-direction – Ändert die Richtung der Elemente (z.B. row für horizontal oder column für vertikal).
